## Description

Second test task based on [Nest](https://github.com/nestjs/nest).

## Installation

```bash
$ npm install
```

## Running the app
Please check src/config.ts to avoid possible problems

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Docker compose
```
$ docker compose up
```
